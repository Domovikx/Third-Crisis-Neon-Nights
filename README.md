# Third Crisis: Neon Nights — Русская локализация

> Инструментарий для извлечения, классификации и перевода текста игры **Third Crisis: Neon Nights** (Anduo Games).
> Парсинг Unity serialized данных (версия 22) на чистом Node.js — без единой сторонней библиотеки.

**Поисковые теги:** `Third Crisis Neon Nights` `русская локализация` `русификатор` `Russian translation` `Anduo Games` `Unity 2022.3` `PlayMaker` `визуальная новелла` `киберпанк` `перевод игры` `Node.js` `NDJSON` `l10n` `i18n` `BepInEx`

**38 исходных файлов**, **2,2M+ сырых строк**, **24K+ диалогов**, **758K+ UI-элементов**.

### Этапы

1. **parser.mjs** — читает бинарные файлы игры (level0-15, sharedassets0-15, resources.assets, globalgamemanagers, DLL). Извлекает **null-terminated ASCII** (data-секция) + **aligned strings** (length-prefixed) + **UTF-16 LE** (метаданные .NET сборок). Никаких сторонних библиотек.
2. **bundle-parser.mjs** — читает Addressables .bundle файлы (UnityFS v8, LZ4HC сжатие), декомпрессирует заголовок, сканирует несжатую data-область на ASCII строки.
3. **extractor.mjs** — принимает NDJSON от парсера, классифицирует строки (dialogue / UI / noise), разбивает по source-файлам, выдаёт NDJSON для перевода.

## Быстрый старт

```bash
npm run parse          # парсер (binary → NDJSON, 36-38 файлов)

npm run test:parse     # тесты парсера (72 теста)
npm run test:bundle    # тесты bundle-парсера
```

## Ключевые открытия (технические)

### UTF-16 LE строки в DLL

Часть UI-текста настроек (Resolution Scaling, Environment Effects, Realtime Reflections, Post Processing) хранится не как `Settings.*` ключи в resources.assets, а как **UTF-16 LE строки** внутри сериализованных данных Assembly-CSharp.dll. Парсер извлекает их через двухпроходное сканирование `(printable_ascii, 0x00)` пар.

### NToolkit type-tag сериализация

Настройки Video-таба сериализованы с префиксными type-тегами:

```
Settings.Fullscreen → \x15 Fullscreen
Settings.Resolution → \x25 Resolution Scaling
                      \x27 Environment Effects
                      \x29 Realtime Reflections
                      \x1f Post Processing
Эти 4 — sub-опции без собственных Settings.* ключей.
```

### Data-секция бандлов не сжата

У Addressables бандлов (UnityFS) сжат только заголовок. Data-область — raw бинарная память, читаемая напрямую. Из 97 бандлов только 4 содержат UI текст: level-cartelhideout, level-glowinghole, 3dsuitcasescene, releasenotesui.

## Формат NDJSON

**Parser (вход для экстрактора):**

```json
[offset,"raw"]
[1131336,"In final Room Location Nova"]
[2355047,"Resolution Scaling"]
```

**Extractor (для перевода):**

```json
["{source}_{seq}","{original}","{translated}","{offset}"]
["level3_001","And now hold still I'm not done yet.","","641239"]
["Assembly-CSharp_1735","Resolution Scaling","","2355047"]
```

Третий элемент заполняется при переводе. offset сохраняется для обратной записи в бинарник.
