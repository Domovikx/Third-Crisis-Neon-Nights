# Third Crisis Neon Nights — Русский перевод

Автоматический перевод игры на русский язык с использованием [XUnity Auto Translator](https://github.com/bbepis/XUnity.AutoTranslator).

## Быстрая установка

Требования: [Node.js](https://nodejs.org/)

```bash
node install-translation.mjs
```

Скрипт автоматически скачает и установит BepInEx + XUnity Auto Translator.

## Установка вручную

### 1. Скачивание

- **BepInEx 5.4**: [GitHub Releases](https://github.com/BepInEx/BepInEx/releases/latest)
  - Файл: `BepInEx_win_x64_5.4.XX.X.zip`

- **XUnity Auto Translator**: [GitHub Releases](https://github.com/bbepis/XUnity.AutoTranslator/releases)
  - Файл: `XUnity.AutoTranslator-BepInEx-{VERSION}.zip`

### 2. Установка

1. Распаковать BepInEx в папку игры
2. Распаковать XUnity Auto Translator поверх (в папку игры)

### 3. Настройка

Конфиг: `BepInEx/config/AutoTranslatorConfig.ini`

```ini
[Service]
Endpoint=GoogleTranslateV2

[General]
Language=ru
FromLanguage=en

[Http]
DisableCertificateValidation=True
```

### 4. Запуск

Запустить игру. Консоль откроется вместе с игрой.

## Горячие клавиши

| Клавиша | Действие |
|--------|----------|
| `ALT+0` | UI переводчика |
| `ALT+T` | Вкл/выкл перевод |
| `ALT+R` | Перезагрузить перевод |
| `ALT+U` | Ручной перевод |

## Переводчики

| Переводчик | Ограничения | Качество |
|-----------|-------------|----------|
| GoogleTranslateV2 | Бесплатно, может сломаться | Среднее |

## Troubleshooting

Если не переводит:
1. Проверь консоль на ошибки
2. Убедись что `DisableCertificateValidation=True` в секции `[Http]`
