## Описание
Пакетный перевод непереведённых английских строк на русский через Google Translate API.
Результаты сохраняются прямо в `_AutoGeneratedTranslations.txt`.

## Когда использовать
- Когда нужно быстро перевести большой объём непереведённых строк
- После первичного анализа (скил `translate-analysis`)

## Скрипт
```bash
node .opencode/skills/translate-batch/batch.mjs              # перевести непереведённое
node .opencode/skills/translate-batch/batch.mjs --source файл # перевести строки из файла
node .opencode/skills/translate-batch/batch.mjs --dry-run     # тестовый прогон
```

## Важно
- Переводит пачками по 25 строк с паузой 1с между пакетами
- Сохраняет прогресс после каждого пакета
- Использует бесплатный Google Translate (без API ключа)
