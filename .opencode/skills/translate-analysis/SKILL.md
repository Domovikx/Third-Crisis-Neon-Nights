## Описание
Анализирует файлы перевода `_AutoGeneratedTranslations.txt` в `BepInEx/Translation/ru/Text/`.
Считает статистику, находит непереведённые строки, проверяет качество.

## Когда использовать
- Когда нужно оценить объём непереведённого текста
- Для проверки качества машинного перевода
- Для сравнения en-кэша и ru-перевода

## Скрипт
В папке скила лежит `analyze.mjs`. Запуск:
```bash
node .opencode/skills/translate-analysis/analyze.mjs
```

## Структура перевода
Формат: `оригинал=перевод`
Файлы:
- `BepInEx/Translation/ru/Text/_AutoGeneratedTranslations.txt` — перевод
- `BepInEx/Translation/en/Text/_AutoGeneratedTranslations.txt` — кэш оригиналов
- `_Substitutions.txt` — точные замены
- `_Preprocessors.txt` — предобработка
- `_Postprocessors.txt` — постобработка
