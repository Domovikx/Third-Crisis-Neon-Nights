# Назначение

Этот скил запускает параллельную обработку нескольких файлов с помощью `task(general, ...)`.
Каждый файл получает отдельного `general`-агента со своей изолированной сессией.

## Входные данные

Список путей к файлам

## Процесс

1. Для КАЖДОГО файла из списка запусти ОДНИМ СООБЩЕНИЕМ (одной пачкой):
   task(general, "Переведи файл {file} на русский.

   Инструкция:
   - Загрузи скил translate-game
   - Переведи файл используя скил translate-game построчно, в конце проверь качество перевода всего файла скилом translate-game с целью улучшения качества и выявления ляпов
     ")

2. Дождись результатов от всех task(general)
3. Верни итоговый отчёт

## Ограничения

- Не более 5 параллельных task(general) одновременно
- Каждый general работает независимо в своей сессии
